ATTRIBUTE SELECTORS

[attribute] Selector

The links with a target attribute gets a cyan background:

w3schools.com disney.com wikipedia.org

[attribute="value"] Selector

The link with target="_blank" gets a lightblue background:

w3schools.com disney.com wikipedia.org

[attribute~="value"] Selector

All images with title~="flower" the title attribute containing the word "flower" get a cyan border. Not hyphen (-)

[attribute|="value"] Selector

All element with class|="top" the top attribute get a cyan background. The value can be exactly specified value, or specified value followed by a hypen (-)
